Portfolio management-integrate TFS with Project Server

2010, Dec 09 1 min read

Finally the beta of integration between project server and TFS is out, and you can try it. All the detail can be found in the post in Brian Harry’s blog. Basically is a tool that manage synchronization between a TFS instance and project server, that permits to managers to work on project server for resource management, while developers still use TFS for agile process management.

If you have project server and tfs in your organization, this is a tool you should not miss.
